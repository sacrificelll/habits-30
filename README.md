# Habits 30 — трекер привычек на 30 дней

Простое веб-приложение, чтобы вырабатывать привычки: заводишь до 5 привычек, каждый день одним кликом отмечаешь выполнение, видишь календарь прогресса, серии подряд (streak) и общую статистику. Без регистрации и сервера — данные хранятся прямо в браузере.

🔗 **Демо (Vercel):** _ссылка появится после деплоя_
📹 **Видео-разбор:** _ссылка появится к сдаче_

> Тестовое задание для OPERIX. Стек: Next.js + TypeScript + Tailwind CSS.

---

## Возможности (из ТЗ)

- ✅ До 5 привычек: название, цель в днях (7–30), цвет
- ✅ Отметка выполнения за день в один клик/тап
- ✅ Календарь прогресса — сетка 30 дней по каждой привычке
- ✅ Текущий streak и лучший streak
- ✅ Общая статистика: процент выполнения, всего отмеченных дней
- ✅ Сброс данных одной кнопкой с подтверждением
- ✅ Мобильная адаптация
- ✅ Деплой на Vercel

## Как запустить локально

> _Уточнится после создания проекта. Ориентировочно:_

```bash
git clone <ссылка-на-репозиторий>
cd habits-30
npm install
npm run dev
# открыть http://localhost:3000
```
